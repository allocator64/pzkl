# PZKL task
## Building

1. Install Protobuf and cmake from desired sources
2. Get current version of mystem from https://tech.yandex.ru/mystem/ and put binary in mystem/ directory.
3. Execute:
```
# cd build
# cmake ..
# cd ../
# make -C build
```

## Running

1. Put your text in texts/input.txt
2. Execute:
```
# build/main
```
